## 프롤로그

VoIP 및 통합 커뮤니케이션 분야에서 가장 많이 사용되는 프로토콜은 SIP이다. SIP 프로토콜은 VoIP 프로토콜에서 대세로 자리 잡으면서 전화기, 음성 게이트웨이, IP PBX, SBC 등에서 광범위하게 사용된다. SIP 프로토콜은 인터넷 전화나 영상 회의 솔루션을 다루는 엔지니어들이 반드시 공부해야 하는 프로토콜이다.
