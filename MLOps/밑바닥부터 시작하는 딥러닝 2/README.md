## 책에서 배우는 내용

```
파이썬을 이용한 텍스트 처리
딥러닝 등장 이전의 단어 표현 방법
단어 벡터를 얻기 위한 word2vec (CBOW 모델과 skip-gram 모델)
대규모 데이터의학습을 고속화하는 네거티브 샘플링
시계열 데이터를 처리하는 RNN, LSTM, GRU
시계열 데이터의 오차역전파법인 BPTT
문장을 생성하는 신경망
시계열 데이터를 다른 시계열 데이터로 변환하는 seq2seq
중요 정보에 주목하는 어텐션
```
