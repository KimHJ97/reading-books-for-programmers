# 참고사항

 - `FastAPI 서버 실행`
    - backend 디렉토리로 이동한 후 아래 명령어를 실행한다.
```bash
(venv) uvicorn main:app --reload
```

<br/>

 - `Svelte 서버 실행`
    - frontend 디렉토리로 이동한 후 아래 명령어를 실행한다.
```bash
$ npm run dev
```

<br/>

 - `alembic`
```bash
(venv) alembic init migrations
(venv) alembic revision --autogenerate
(venv) alembic upgrade head
```
